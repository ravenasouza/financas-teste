# Lumina: Agente Financeiro Inteligente

A **Lumina** é um protótipo de agente de IA Generativa focado em relacionamento financeiro proativo. O projeto foi desenvolvido para transformar dados bancários frios em uma experiência consultiva e motivadora para o usuário.

---

## Caso de Uso

O projeto foca no cliente **João Silva**, um Analista de Sistemas que busca comprar um apartamento em 2027.  
A Lumina analisa suas transações e perfil para oferecer **insights reais e seguros**.

---

## Tecnologias

- **Linguagem:** Python 3.10+
- **Interface:** Streamlit (foco em UX)
- **Dados:** Pandas & JSON
- **IA:** Simulação de IA Generativa com foco em anti-alucinação

---

## Estrutura do Repositório

- `data/` → Base de conhecimento (CSV/JSON)  
- `docs/` → Documentação do agente, prompts e métricas  
- `src/` → Código-fonte da aplicação  

---

## Como Rodar o Projeto

### 1. Clone o repositório
```bash
git clone https://github.com/seu-usuario/lab-agente-financeiro.git
```

### 2. Instale as dependências
```bash
pip install streamlit pandas
```

### 3. Execute a aplicação:
```bash
streamlit run src/app.py
```
